# Seam

A personal sewing companion app. Track your projects, fabric stash, patterns, body measurements, and finished makes — all connected, all in one place.

## What it does

- Capture project ideas quickly before they're forgotten
- Log your fabric stash with photos, properties, and reservation status
- Link fabrics, patterns, and sketches to projects
- Store body measurements and fitting notes
- Archive finished makes with lessons learned
